Word Embeddings and Self-Organizing Maps

A Skip-gram embedding model paired with a self-organizing map, turning movie-review text into vector neighborhoods and visual word clusters.

View GitHub Read Notebook Open in Colab

At A Glance

141Saved word vectors used in the notebook visualization.
SOMCustom best-matching unit search, neighborhood update, and U-matrix tools.
NLPText preprocessing, Skip-gram training, similarity search, and word-cloud analysis.

Pipeline

1. Tokenize textClean IMDb-style reviews and map words to integer ids.
2. Train Skip-gramPredict context words from target words to learn embedding vectors.
3. Search neighborsUse cosine similarity to find words that live nearby in embedding space.
4. Map structureTrain a SOM to place high-dimensional word vectors onto a 2D lattice.

The SOM is useful because it makes embedding neighborhoods inspectable. Similar words can be highlighted on a learned 2D grid, which turns a high-dimensional vector model into something easier to reason about.
