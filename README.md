# CS 443 Project 3: Word Embeddings and Self-Organizing Maps

Completed CS 443 Bio-inspired Machine Learning project focused on word embeddings and self-organizing maps.

This project includes:

- Skip-gram word embedding model code
- text preprocessing helpers for IMDb-style review data
- self-organizing map implementation
- completed Jupyter notebooks for word embeddings and word-cloud/SOM analysis
- saved small artifacts used by the notebooks

## Files

- `word_embeddings.ipynb` walks through text preprocessing, Skip-gram training, and embedding analysis.
- `word_cloud.ipynb` uses trained embeddings and SOM-style visualization.
- `skipgram.py`, `skipgram_layers.py`, `som.py`, and `text_dataset_word.py` contain the main implementations.
- `data/imdb_train.csv` is a small included dataset for reproducible notebook runs.
